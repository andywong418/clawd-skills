# Quality Prompt Template

Append this to all UGC creator image prompts for maximum realism:

```
Transform the image into a high-quality, professional photograph as if captured today. Style: A candid, unfiltered iPhone 15 Pro Max photo. Lens: Shot with a 24mm lens at f/11 so the background is fully sharp and in focus (no blur). Quality: The image is 100% clean—no text, no letters, no grain, zero blur, and no filters. It looks like a high-quality snapshot taken just now. Lighting: Natural Window Light with High Dynamic Range (HDR) for perfectly balanced highlights and shadows.
```
